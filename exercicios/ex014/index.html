Capítulo 1 CSS Internal Style

Cap 1.1

O Estilo Interno é feito usando a tag CSS style na area do Head, todos os estilos são feitos dentro desta tag. É melhor usar esse modo de style do que o CSS inline. Fica bem mais organizado. Desvantagens: Ocupa mais espaço do que o próprio conteúdo HTML; Precisaremos copiar e colar em todas as páginas que você vai usar. Apenas para uma página é okay, mais de uma página não é recomendado, use CSS External.
